ukulele.js - 尤克里里指位系统
核心逻辑：基于C调指位数据库，通过半音偏移量动态计算任意调性的音符指位，并提供UI显示和可视化。
一、主要功能模块：
1.静态指位数据库
C_POSITIONS: 定义所有音符在C调中的标准指位（弦数1-4，品数0-18）
KEY_OFFSETS: 各调性相对于C调的半音偏移量映射
2.智能指位计算引擎
calculatePositionsBidirectional(): 核心算法，基于C调指位+调性偏移量，双向推导（升调/降调）所有可能的有效指位
precalculateAllPositions(): 启动时预计算12个调的全部指位并缓存，运行时直接查询
3.UI信息更新
updateUkulelePosition(): 根据当前音符和调性：
-自动转换升降号表示（按调性习惯显示）
-分离常用指位（0-5品）和高把位（6品+）
-显示相关和弦推荐
-格式化输出为"X弦Y品"中文描述
4.交互与可视化
-调性选择器联动：切换调性时自动刷新当前音符指位
-visualizeUkulelePositions(): 可选功能，生成指板网格图，直观显示所有位置
二、技术特点：
1.空间换时间：预计算缓存避免重复计算
2.双向推导：同时考虑升降调指位等价性（C# = Db）
3.调性感知：根据当前调性自动选择最合理的音符表示法
4.模块化设计：计算、UI、可视化分层解耦
