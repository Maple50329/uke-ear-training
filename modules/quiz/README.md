答题流程与数据统计核心模块梳理：
1. history.js
功能：历史记录管理与回放系统
自动记录：答对题目后自动保存（含基准音、目标音、调性、唱名）
双向同步：桌面端和移动端历史列表实时同步
点击回放：支持点击历史项重播基准音/目标音
持久化：localStorage 存储，关闭浏览器不丢失（上限 30 条）
唱名识别：自动从答题按钮提取唱名（如"Do→Mi"）
2. manager.js
功能：题目播放与答题检查**（整个应用的核心引擎）**
完整流程：音阶 → 基准音×2 → 目标音（可配置跳过音阶）
智能选题：避免与最近 3 题重复，确保练习多样性
预选模式：调性/难度/基准音模式在播放中更改 → 下题生效
答题逻辑：
答对 → 禁用答题区、显示音程/尤克里里信息、触发自动下一题
答错 → 保留答题区、记录尝试次数、可无限重试
状态联动：答对后触发 answer-correct 事件，驱动状态栏更新
3. reset-manager.js
功能：重置逻辑与状态清理
三重重置：
1.轻量重置：保留当前设置，仅清理答题状态（欢迎界面）
2.完整重置：清理所有预选、定时器、统计，回到初始状态
3.自动重置：开启"自动下一题"时，答对后自动开始新题
安全停止：强制停止 Tone.js Transport、清理音频节点
预选清理：防止重置后调性/难度/基准音模式残留
4. stats-manager.js
功能：练习数据统计与分析**（最复杂的模块）**
今日统计：练习次数、一次性答对、重试答对、正确率
历史统计：总题目数、总正确数、总正确率（与今日区分）
连胜系统：连续练习天数、答题连胜（当前/最高）
分类统计：基准音（C/A）、调性（C-G）、难度（basic/extended）、音级类型（自然/变化）
数据持久化：localStorage 存储，每日自动重置今日数据
最佳追踪：自动记录并更新历史最高正确率
数据流关系：
manager.js 答对 → addToHistory() 记录历史 → statsManager.recordAnswer() 更新统计 → history.js 渲染列表 → stats-manager.js 刷新右侧面板显示
