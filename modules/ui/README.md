1.answer-grid.js: 答题按钮网格系统 
- 动态生成8键/13键答题按钮（Do/Re/Mi...），处理按钮点击事件、视觉反馈（播放状态高亮）、响应式缩放和八度自适应。

2.buttons.js: 主控制按钮
- 管理大播放按钮状态（播放/重播/下一题/音阶/基准音），空格键快捷播放，R键复位题目，与答题锁定状态同步。

3.feedback.js: 答题反馈系统 
- 显示对错动画、更新音程分析面板、尤克里里指位图、音高/频率/调式级数实时显示，控制答题区禁用状态。

4.range-manager.js: 音域管理 
- 切换"C3-B3小字组"和"C4-B4小字一组"，处理播放中/答题后等边界情况下的待处理更改，同步UI状态。

5.settings.js: 设置控件 
- 初始化"自动下一题"时长滑块（3-5秒），基准音模式按钮组（固定C/固定A）的交互逻辑，移动端侧边栏控制。

6.theme-manager.js: 主题控制 
- 深色/浅色模式切换，localStorage记忆，自动跟随系统主题，CSS变量驱动。

7.panel-manager.js: 桌面端浮动面板 
- 可拖拽的"音高-音程-尤克里里"三页卡片，支持滑动切换、智能贴边吸附、位置记忆。

8.mobile-panels.js: 移动端面板管理器 
- 左右滑出式设置/统计面板，手势滑动关闭（需阈值60px），双向同步桌面端数据到移动端。

9.mobile-content-loader.js: 移动端内容加载 
- 动态填充移动端设置面板（音域、基准音、调性、难度），实现移动端与桌面端设置的实时双向同步。

10.modal-manager.js: 音域选择模态框 
- 居中弹出的音域选择器，点击确定后应用更改，背景遮罩层点击关闭。

11.stats-modal.js: 详细统计模态框 
- 单例模式，显示总体/今日/分类统计（基准音/音级类型/调性/难度），支持数据重置和导出提示。

12.event-bindings.js: 全局事件绑定 
- 主题切换、基准音模式按钮组、调性选择器的统一事件注册，清理旧监听器防止重复绑定。

13.status-bar.js: 桌面端底部状态栏 
- 固定显示当前基准音模式、调性、难度、音域、播放状态，实时监听15+种状态变化事件自动更新。

整体架构特点: 
采用AppState集中状态管理 + AppGlobal工具箱模式，严格区分桌面端和移动端逻辑，响应式设计，手势交互完善，边界条件处理细致（播放中禁止修改设置等）。
